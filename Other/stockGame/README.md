# Stock Trading Game

A browser-based stock market simulation where you trade fictional stocks across three sectors, react to breaking news events, and try to grow your portfolio.

## Features

- **10 stocks** across Tech, Healthcare, and Energy sectors
- **Realistic price movement** with hidden trends, momentum, and volatility
- **Dynamic news events** that impact individual stocks or entire sectors
- **Bankruptcy system** — stocks can fail, wiping out your shares
- **Adjustable speed** and pause/play controls
- **Live charts** with mini sparklines and a detailed view for selected stocks

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- TypeScript
- Vite
